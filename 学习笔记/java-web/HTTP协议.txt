响应体:
    1.响应体就是响应的消息体，如果是纯数据就是返回纯数据，
        比如@ResponseBody Controller方法中返回字符串数据,页面上就只显示这个字符串
    2.如果请求的是HTML页面，那么返回的就是HTML代码(所以为什么说后端要渲染页面!?懂了吧???)
        当你点击链接的时候,会跳转到一个新页面.这个新页面的html代码是怎么生成的呢?
                首先,这个新页面并不是你用模板引擎(JSP,ThymeLeaf等)写好的那个页面!
                而是在后端把模板页面填充数据后生成的新的html代码,并装进响应体中返回浏览器,再经过CSS渲染后,最终呈现出你看到的页面!
        所以说: 程序员用模板引擎写的jsp/html页面,真的就只是模板而已!!!实际上你在浏览器看到的页面是后台生成html代码并通过响应体发送过来的!
