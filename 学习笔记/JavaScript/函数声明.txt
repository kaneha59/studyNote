js中声明函数有两种常用方式.
    // 函数表达式
    var f = function() {
        console.log(1);  
    }

    // 函数声明
    function f (){
        console.log(2);
    }


函数表达式: 也可以说是匿名函数.无需声明函数名,一般赋值给一个变量用于调用.函数表达式可以存储在变量或者 对象 属性中
    声明: var <变量名> = function() {...}
    函数名: 一般不添加函数名
    函数提升: 否

函数声明: 具名函数,需要声明函数名.
    声明: function <函数名>() {...}
    函数名: 需要定义函数名
    函数提升: 是.
